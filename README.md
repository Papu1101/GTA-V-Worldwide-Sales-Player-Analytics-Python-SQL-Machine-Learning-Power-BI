# 🎮 GTA V Worldwide Sales & Player Analytics

## 📌 Project Overview

This project presents an end-to-end data analytics workflow using Grand Theft Auto V worldwide sales and player engagement data from 2013 to 2026. The dataset contains nearly 15,000 monthly records across 88 countries and 7 gaming platforms, covering sales performance, customer behavior, player engagement, marketing campaigns, and macroeconomic indicators.

The objective is to transform raw gaming data into meaningful business insights using Python, SQL, Machine Learning, and Power BI.

---

## 🎯 Business Objectives

- Analyze worldwide sales performance
- Understand customer acquisition and retention
- Evaluate platform and regional performance
- Measure player engagement across platforms
- Assess marketing campaign effectiveness
- Identify revenue drivers and business trends
- Build predictive machine learning models
- Develop interactive Power BI dashboards
